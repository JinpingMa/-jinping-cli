# install-lint-config

> 自动安装统一的lint规则文件，可选择eslint、stylelint

## 使用

* 不安装使用
```
  > npx git+https://github.com/JinpingMa/install-lint-config.git
```
* 安装使用
```
  > npm install git+https://github.com/JinpingMa/install-lint-config.git
  > install-lint-config
```
* 选择要安装的lint文件，可多选。

## 参数列表

* `-all or -a`: 安装全部lint文件
